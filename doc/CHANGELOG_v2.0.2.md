# v2.0.2 更新日志

发布日期：2025-01-02

## ⚡ 性能优化

### ONNX Runtime 推理优化
- **YOLO OBB 检测器 CUDA 加速修复**：移除导致 Fallback 模式的配置，推理速度从 1.2+ 秒优化到 0.3 秒（提升 4 倍）
- **统一 ONNX Runtime 配置**：为所有 ONNX 模型设置统一日志级别，隐藏无害警告
- **PaddleOCR 分批处理**：添加分批处理（每批 16 个），避免大图内存峰值

### 日志输出优化
- 删除调试日志，过滤无害警告，输出更简洁

## 🐛 修复

### 替换翻译模式异常处理修复
- 修复嵌套异常处理导致的 `UnboundLocalError` 错误
- 添加空区域检查，避免 `ValueError` 错误

### Photoshop 导出兼容性修复
- 修复 Photoshop 2020+ 版本打开纯色 PNG 时无法识别背景层的问题

## ⚡ 优化

### 替换翻译模式图像合成算法升级
- 新增高级图像合成算法，使用色阶调整 + 正片叠底融合，文字边缘更自然

### Qt UI 优化
- 移除翻译器强制密钥要求，支持本地模型无需密钥
- 禁用主页下拉菜单的鼠标滚轮事件
- 超分倍率下拉框最小宽度设置为 100 像素
