# v2.0.5 更新日志

发布日期：2026-01-05

## 🐛 修复

- 修复 UI 初始化时配置加载重复导致控件重复创建的问题
- 修复配置加载时下拉框触发不必要的配置保存
- 修复"忽略气泡"功能导致气泡区域被膨胀变大的问题
- 修复替换翻译模式下一个大翻译框匹配多个小生肉框时只保留一个的问题
- 修复替换翻译逻辑中的重复 return 语句和 texts 数组重建问题
- 修复译文修改时切换到下一条不点应用导致前一条自动还原的问题
- 修复分镜检测中线段几何计算导致的崩溃问题（添加除零和无效数据保护）
- 修复停止翻译时按钮状态混乱的问题（添加任务ID机制）
- 修复替换翻译模式下修复模型为 none 时蒙版过大的问题（简化为二值化+膨胀）

## ✨ 新增功能

- 新增"翻译完成后卸载模型"配置选项（基础设置 - CLI 区域）
- 增强内存清理：显式卸载模型实例、清理 PyTorch 内部缓存

## 🔧 优化

- 统一保存和清理逻辑，保存文件后立即释放内存，改善超分等大图处理的内存占用
