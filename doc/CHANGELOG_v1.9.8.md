# v1.9.8 更新日志

发布日期：2025-12-22

## ✨ 新功能

### 编辑器功能
- **新增 JSON 保存功能**：编辑器现在支持保存当前编辑状态为 JSON 文件，方便后续恢复和分享
- **切换文件提示框适配 JSON 保存**：切换文件时的未保存提示框新增"保存JSON"选项，用户可选择导出图片、保存JSON或不保存

## ⚡ 优化

### 文本渲染
- **"AI断句居中"选项重命名为"垂直居中"**：更准确地反映其功能，该选项现在仅控制文本在气泡中的垂直居中，横排文本保持默认水平居中

### 编辑器优化
- **移除"生成预览"按钮**：简化编辑器界面，提升性能

## 🐛 修复

### 检测与蒙版修复
- **修复高分辨率检测时蒙版错位问题**：修复当 `detection_size` 设置为 3000 等较大值时，文本蒙版与检测框不对齐导致文本无法正确擦除的问题。原因是去除 padding 时使用 `elif` 导致水平和垂直方向的 padding 无法同时处理，现已修复为同时处理两个方向的 padding，并确保蒙版正确缩放到原图尺寸

### 并行模式修复
- **修复并行模式不保存修复缓存的问题**：修复在并行处理模式下，修复后的图片缓存未正确保存的 bug

### 配置修复
- **修复过滤列表开关失效问题**：修复关闭"启用过滤列表"开关后，过滤功能仍然生效的 bug
- **修复.env编辑切换预设时预设未保存的问题**：修复在.env编辑界面切换预设时，当前预设的修改未保存就切换到新预设的 bug
